Adds the widget to show logged In user's recent search history.

== Description ==
If you have a Search box on the site,
this plugin automatically records what logged In user's are searching and keep record.
simply displays the recent searches of logged In User. 


== Installation ==
Install the plugin from the 'Plugins' section in your dashboard (Go to `Plugins > Add New > Search` and search for User recent search history).
Activate it through the 'Plugins' section.


= Widgets: User's recent Search History =
Displays the recent search history of Logged In User.
